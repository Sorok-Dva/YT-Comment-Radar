FAQ clé API & quota

L’aide essentielle est maintenant visible directement ici.

Où créer la clé API ?

Dans Google Cloud Console : créez ou sélectionnez un projet, activez YouTube Data API v3, puis allez dans APIs & Services > Credentials et créez une API key.

Où récupérer une clé déjà créée ?

Ouvrez le bon projet Google Cloud, allez dans APIs & Services > Credentials, ouvrez la clé voulue et copiez sa valeur.

Pourquoi une clé neuve peut afficher “quota exceeded” ?

Parce que le quota est lié au projet Google Cloud, pas à la clé. Une clé neuve dans un projet déjà vidé hérite du même quota déjà consommé.

Solution rapide si le quota est mort ?

Créez un nouveau projet Google Cloud, activez YouTube Data API v3, puis créez une nouvelle clé dans ce nouveau projet.

Quand le quota se réinitialise ?

Le quota quotidien YouTube Data se réinitialise à minuit, heure du Pacifique. Les requêtes invalides consomment aussi du quota.

J’ai leak ma clé, je fais quoi ?

Faites une rotation ou supprimez-la dans Google Cloud Console, puis remplacez-la dans les réglages de l’extension.
