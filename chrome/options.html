FAQ rapide

Le but est d’éviter le combo classique “j’ai créé une clé, ça ne marche pas, je ne sais plus où cliquer”.

Où créer une clé API YouTube Data ?

Dans Google Cloud Console : créez ou sélectionnez un projet, activez YouTube Data API v3, puis allez dans APIs & Services > Credentials et cliquez sur Create credentials > API key.

J’ai déjà une clé. Où la récupérer ?

Toujours dans APIs & Services > Credentials. Ouvrez la clé voulue, puis copiez la valeur affichée dans le champ de clé API.

Pourquoi une clé neuve peut afficher “quota exceeded” ?

Parce que le quota est attaché au projet Google Cloud, pas à la clé elle-même. Si vous créez une nouvelle clé dans un projet déjà vidé, la nouvelle clé hérite du même quota déjà consommé.

Le plus simple si le quota est mort ?

Créez un nouveau projet Google Cloud, activez YouTube Data API v3, puis créez une nouvelle clé dans ce nouveau projet.

Quand le quota se réinitialise ?

Le quota quotidien YouTube Data se réinitialise à minuit, heure du Pacifique. Les requêtes invalides comptent aussi dans le quota.

Et si j’ai collé ma clé quelque part publiquement ?

Faites une rotation ou supprimez-la dans Google Cloud Console, puis remplacez-la dans les réglages de l’extension.
