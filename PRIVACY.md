# Privacy Policy – YT Comment Radar

_Last updated: [04/26/12026]_

---

## 🇬🇧 English Version

### Overview
YT Comment Radar is a browser extension that notifies users when their YouTube comments receive new likes.

We respect your privacy. This extension is designed to minimize data usage and to operate primarily on the user's device.

---

### Data Collection

The extension does **not collect or store personal data on external servers by default**.

All core functionality operates using data stored **locally in the user's browser**.

---

### What Data is Stored Locally

The extension stores the following data locally:

- YouTube comment identifiers (commentId)
- Associated video identifiers (videoId)
- Last known like counts
- User settings (e.g. scan frequency, preferences, API key)

This data is required to:
- track comment activity
- detect like count changes
- trigger notifications

---

### API Usage

If the user provides a YouTube Data API key:

- It is stored locally in the browser
- It is used only to query YouTube’s official API
- It is never transmitted to any third-party server

---

### Analytics & Telemetry (Optional)

The extension includes an **optional telemetry feature**.

If explicitly enabled by the user, the extension may send **anonymous and aggregated usage metrics** to a server controlled by the developer (yrc.sorokdva.eu) for the sole purpose of improving the extension.

This may include:

- number of active users
- number of scans performed
- number of imports
- number of notifications triggered
- volume of tracked comments

Important:

- This feature is **disabled by default**
- No personally identifiable information is collected
- No comment content, video content, or user identity is transmitted
- Data is aggregated and cannot be used to identify individual users

Users can enable or disable this feature at any time in the extension settings.

---

### Permissions Usage

The extension requires the following permissions:

- **storage**: to store tracked comments and user settings locally
- **alarms**: to schedule periodic scans for updates
- **notifications**: to notify users when new likes are detected
- **tabs / scripting**: to access YouTube pages when importing comments or using fallback methods
- **host permissions (YouTube domains only)**: to interact with YouTube pages required for the extension functionality

---

### Data Sharing

- No personal data is sold or shared
- No personal data is transmitted to third parties
- All core data remains on the user’s device

---

### Third-Party Services

The extension interacts only with:

- YouTube (via official API or website)

No other third-party services are used, except optional telemetry as described above.

---

### Security

All locally stored data remains within the user's browser.

No external database or persistent backend is used for core functionality.

---

### Changes to This Policy

This policy may be updated if the extension evolves.

Users are encouraged to review it periodically.

---

sorokdva.developer@gmail.com

## 🇫🇷 Version Française

### Vue d’ensemble
YT Comment Radar est une extension navigateur qui notifie l’utilisateur lorsque ses commentaires YouTube reçoivent de nouveaux likes.

Nous respectons votre vie privée. Cette extension est conçue pour minimiser l’utilisation de données et fonctionner principalement sur l’appareil de l’utilisateur.

---

### Collecte de données

L’extension **ne collecte ni ne stocke de données personnelles sur des serveurs externes par défaut**.

Toutes les fonctionnalités principales reposent sur des données stockées **localement dans le navigateur**.

---

### Données stockées localement

L’extension stocke localement :

- Identifiants de commentaires YouTube (commentId)
- Identifiants de vidéos associées (videoId)
- Nombre de likes connus
- Paramètres utilisateur (fréquence de scan, préférences, clé API)

Ces données servent à :

- suivre l’activité des commentaires
- détecter les nouveaux likes
- déclencher des notifications

---

### Utilisation de l’API

Si l’utilisateur fournit une clé API YouTube :

- Elle est stockée localement
- Elle est utilisée uniquement pour interroger l’API officielle YouTube
- Elle n’est jamais transmise à un serveur tiers

---

### Analytics & Télémétrie (Optionnelle)

L’extension propose une fonctionnalité de télémétrie **optionnelle et désactivée par défaut**.

Si activée explicitement par l’utilisateur, l’extension peut envoyer des **métriques d’utilisation anonymes et agrégées** vers un serveur contrôlé par le développeur (yrc.sorokdva.eu), uniquement dans le but d’améliorer l’extension.

Cela peut inclure :

- nombre d’utilisateurs actifs
- nombre de scans effectués
- nombre d’imports
- nombre de notifications déclenchées
- volume de commentaires suivis

Important :

- Cette fonctionnalité est **désactivée par défaut**
- Aucune donnée personnelle identifiable n’est collectée
- Aucun contenu de commentaire, de vidéo ou identité utilisateur n’est transmis
- Les données sont agrégées et ne permettent pas d’identifier un utilisateur

L’utilisateur peut activer ou désactiver cette fonctionnalité à tout moment dans les paramètres de l’extension.

---

### Utilisation des permissions

L’extension utilise les permissions suivantes :

- **storage** : stockage local des données
- **alarms** : planification des scans automatiques
- **notifications** : affichage des alertes de nouveaux likes
- **tabs / scripting** : accès aux pages YouTube pour l’import et le fallback
- **host permissions (YouTube uniquement)** : accès limité aux pages YouTube nécessaires

---

### Partage des données

- Aucune donnée personnelle n’est vendue ou partagée
- Aucune donnée personnelle n’est transmise à des tiers
- Toutes les données principales restent sur l’appareil de l’utilisateur

---

### Services tiers

L’extension interagit uniquement avec :

- YouTube (API officielle ou site web)

Aucun autre service tiers n’est utilisé, à l’exception de la télémétrie optionnelle décrite ci-dessus.

---

### Sécurité

Toutes les données restent stockées dans le navigateur de l’utilisateur.

Aucune base de données externe n’est utilisée pour le fonctionnement principal.

---

### Modifications

Cette politique peut évoluer en fonction des mises à jour de l’extension.

Nous recommandons de la consulter régulièrement.
